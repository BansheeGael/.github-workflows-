# Area Dig Mod for Minecraft 1.7.10

**Автор:** Banshee  
**Версия:** 1.0  
**Для Minecraft:** 1.7.10

Мод для копки области 7×7 блоков.

## Функции
- Копка области 7×7 блоков за один клик
- Включение/выключение по клавише F2
- Случайные задержки между блоками для обхода античитов

## Установка
1. Скачать `AreaDig-1.0.jar` из раздела Releases
2. Поместить в папку `.minecraft/mods/`
3. Запустить Minecraft с Forge 1.7.10

## Использование
- **F2** - включить/выключить мод
- **ЛКМ по блоку** - копать область 7x7 вокруг него
